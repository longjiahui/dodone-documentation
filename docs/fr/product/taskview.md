# Vue des tâches

Les vues de tâches permettent de personnaliser les requêtes de tâches et ne stockent pas réellement les données des tâches. Elles se divisent en vues automatiques et vues manuelles. Les vues automatiques possèdent une logique de recherche et parcourent automatiquement la liste des tâches, tandis que les vues manuelles nécessitent d'ajouter les tâches manuellement, semblable à une `étiquette`.

## Propriétés de la vue des tâches

- Nom
- Icône — L'icône d'une vue manuelle s'affichera sur les tâches auxquelles cette vue est assignée. L'icône par défaut pour les vues manuelles est une étiquette, pour les vues automatiques un robot.
- Description
- Type
  - Automatique
  - Manuelle
- Logique des vues automatiques
  - Script
  - Configuration logique assistée par GUI
