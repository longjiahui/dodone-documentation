# Tâche

Les tâches sont les données les plus fondamentales de l'application.

## Propriétés de la tâche

| Champ                  | key         |
| ---------------------- | ----------- |
| id                     | id          |
| contenu                | content     |
| description            | description |
| ordre                  | sortOrder   |
| priorité               | priority    |
| charge de travail      | factor      |
| état d'achèvement      | state       |
| id du groupe           | groupId     |
| id de la tâche parente | parentId    |
| cible                  | target      |
| type de cible          | targetType  |
| date de début          | startAt     |
| date de fin            | endAt       |
| heure d'achèvement     | doneAt      |
| créé le                | createdAt   |
| mis à jour             | updatedAt   |

## Actions sur la tâche

Toutes les actions sur la tâche se trouvent dans la barre d'outils de la tâche. Cliquez sur la zone vide d'une tâche ou sur l'icône plus à droite pour les afficher.

![task-tools](/steps/task/task_tools.png)

De gauche à droite :

- Modifier le contenu de la tâche
- Modifier les sous-tâches
- Modifier la description
- Modifier l'heure
- Modifier la charge de travail
- Modifier la priorité
- Définir la cible
- Déplacer la tâche (changer la tâche parente)
- Définir/annuler l'ancre de tâche
- Gérer les vues manuelles dont fait partie la tâche
- Configurer une tâche récurrente
- Supprimer / (dans une vue manuelle) retirer de la vue
