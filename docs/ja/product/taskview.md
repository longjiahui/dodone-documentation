# タスクビュー

タスクビューはタスクのクエリをカスタマイズでき、実際にタスクデータを保存するものではありません。自動ビューと手動ビューに分かれます。自動ビューはタスク検索ロジックを持ち、タスクリストを自動で検索します。手動ビューは手動でタスクを設定する必要があり、`タグ` 機能に似ています。

## タスクビューの属性

- 名前
- アイコン — 手動タスクビューのアイコンはそのビューが設定されたタスクに表示されます。手動ビューのデフォルトアイコンはタグ、 自動ビューのデフォルトはロボットのアイコンです。
- 説明
- タイプ
  - 自動
  - 手動
- 自動ビューのロジック
  - スクリプト
  - GUI 補助設定ロジック
