# タスク

タスクはアプリケーションで最も基本的なデータです。

## タスクの属性

| フィールド  | key         |
| ----------- | ----------- |
| id          | id          |
| 内容        | content     |
| 説明        | description |
| 並び順      | sortOrder   |
| 優先度      | priority    |
| 作業量      | factor      |
| 完了状態    | state       |
| グループ id | groupId     |
| 親タスク id | parentId    |
| 目標        | target      |
| 目標タイプ  | targetType  |
| 開始日      | startAt     |
| 終了日      | endAt       |
| 完了時刻    | doneAt      |
| 作成時刻    | createdAt   |
| 更新時刻    | updatedAt   |

## タスク操作

すべてのタスク操作はタスクツールバーにあります。タスクの空白部分または右の「その他」アイコンをクリックして表示します。

![task-tools](/steps/task/task_tools.png)

左から右へ：

- タスク内容を編集
- サブタスクを編集
- 説明を編集
- 時間を編集
- 作業量を編集
- 優先度を編集
- 目標を設定
- タスクの位置を移動（親タスクを変更）
- タスクアンカーを設定/解除
- 手動タスクビューを管理
- 繰り返しタスクを設定
- 削除 / （手動ビューでは）ビューから除外
