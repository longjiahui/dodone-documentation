# タスクアンカー

3、4 層以上の多階層のサブタスクを作成すると、タスク管理が煩雑になることがあります。アンカーを作成するとこの問題を解決できます。アンカーはブックマークのようなもので、アンカーを追加するとグループ内から直接アンカーを設定したタスクにアクセスできます。
