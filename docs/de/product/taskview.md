# Aufgabenansicht

Aufgabenansichten ermöglichen es, Abfragen für Aufgaben anzupassen und speichern die eigentlichen Aufgabendaten nicht. Es gibt automatische und manuelle Ansichten. Automatische Ansichten verfügen über eine Suchlogik und durchsuchen die Aufgabenliste automatisch, während manuelle Ansichten manuell mit Aufgaben gefüllt werden und einem `Tag`-System ähneln.

## Eigenschaften der Aufgabenansicht

- Name
- Icon — Das Icon einer manuellen Ansicht wird auf Aufgaben angezeigt, denen diese Ansicht zugewiesen wurde. Das Standardicon für manuelle Ansichten ist ein Tag, für automatische Ansichten ein Roboter.
- Beschreibung
- Typ
  - Automatisch
  - Manuell
- Logik der automatischen Ansicht
  - Skript
  - GUI-unterstützte Logikeinstellungen
