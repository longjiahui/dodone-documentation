# Aufgabe

Aufgaben sind die grundlegendsten Daten in der Anwendung.

## Aufgaben-Eigenschaften

| Feld                | key         |
| ------------------- | ----------- |
| id                  | id          |
| Inhalt              | content     |
| Beschreibung        | description |
| Sortierreihenfolge  | sortOrder   |
| Priorität           | priority    |
| Arbeitsaufwand      | factor      |
| Erledigungsstatus   | state       |
| Gruppen-id          | groupId     |
| Elternaufgabe-id    | parentId    |
| Ziel                | target      |
| Zieltyp             | targetType  |
| Startdatum          | startAt     |
| Enddatum            | endAt       |
| Erledigt am         | doneAt      |
| Erstellungszeit     | createdAt   |
| Aktualisierungszeit | updatedAt   |

## Aufgabenaktionen

Alle Aufgabenaktionen befinden sich in der Aufgaben-Toolbar. Klicken Sie auf den leeren Bereich einer Aufgabe oder das Mehr-Icon rechts, um sie anzuzeigen.

![task-tools](/steps/task/task_tools.png)

Von links nach rechts:

- Inhalt der Aufgabe bearbeiten
- Unteraufgaben bearbeiten
- Beschreibung bearbeiten
- Zeit bearbeiten
- Arbeitsaufwand bearbeiten
- Priorität bearbeiten
- Ziel setzen
- Position der Aufgabe ändern (Elternaufgabe ändern)
- Anker setzen/entfernen
- Manuelle Ansichten verwalten, in denen sich die Aufgabe befindet
- Wiederkehrende Aufgabe einstellen
- Löschen / (in manueller Ansicht) Aus Ansicht entfernen
