# Aktives Fenster

Das aktive Fenster zeigt die Aufgabe an, an der Sie gerade arbeiten, und hat zwei Modi:

- Eine bestimmte Aufgabe im aktiven Fenster anzeigen
- Den Kalender automatisch auslesen, um die aktuelle Aufgabe zu ermitteln

## Aktives Fenster öffnen

### Über die Task-Toolbar öffnen — legt eine bestimmte Aufgabe im aktiven Fenster fest

![open](/steps/doingwindow/open1.png)

### Vom Hauptfenster öffnen — liest automatisch Aufgaben aus dem Kalender
