# Aufgaben-Gruppe

Eine Sammlung von Aufgaben. Alle Aufgaben müssen in einer Gruppe enthalten sein.

## Eigenschaften der Aufgaben-Gruppe

- Name
- Icon (Standardicon ist Ordner)
- Themenfarbe
- Beschreibung
