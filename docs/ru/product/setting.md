## Настройки базы данных

База данных — это файл, в котором хранятся все связанные данные. Создание новой базы данных эквивалентно созданию совершенно нового пространства данных для хранения новых данных. Если предыдущий файл базы данных становится слишком большим и вызывает задержки, вы можете попробовать создать новый файл.

#### Удалить базу данных

:::warning
Удаление базы данных необратимо. Пожалуйста, будьте осторожны.
:::

Нельзя удалить базу данных, которая в данный момент используется. Если вам нужно удалить её, сначала создайте пустую базу данных и переключитесь на неё, затем выполните удаление.

![database](/steps/setting/delete_database.png)
