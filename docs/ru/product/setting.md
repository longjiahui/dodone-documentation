## Настройки базы данных

База данных - это файл, в котором хранятся все связанные данные. Создание новой базы данных равносильно созданию совершенно нового пространства данных для хранения новых данных. Когда предыдущий файл базы данных становится слишком большим и замедляется, вы можете попробовать открыть новый файл.

#### Удалить базу данных

:::warning
Удаление базы данных необратимо. Пожалуйста, действуйте осторожно.
:::

Вы не можете удалить базу данных, которая в настоящее время используется. Если вам нужно ее удалить, сначала создайте пустую базу данных, переключитесь на новую базу данных, а затем приступайте к удалению.

![database](/steps/setting/delete_database.png)
