# Задача

Задачи — это самые базовые данные в приложении.

## Свойства задачи

| Поле                   | key         |
| ---------------------- | ----------- |
| id                     | id          |
| содержимое             | content     |
| описание               | description |
| порядок сортировки     | sortOrder   |
| приоритет              | priority    |
| трудоёмкость           | factor      |
| состояние выполнения   | state       |
| id группы              | groupId     |
| id родительской задачи | parentId    |
| цель                   | target      |
| тип цели               | targetType  |
| дата начала            | startAt     |
| дата окончания         | endAt       |
| время выполнения       | doneAt      |
| время создания         | createdAt   |
| время обновления       | updatedAt   |

## Действия с задачой

Все действия с задачей находятся на панели инструментов задачи. Нажмите на пустую область задачи или на иконку «ещё» справа, чтобы увидеть их.

![task-tools](/steps/task/task_tools.png)

Слева направо:

- Редактировать содержимое задачи
- Редактировать подзадачи
- Редактировать описание
- Редактировать время
- Редактировать трудоёмкость
- Редактировать приоритет
- Установить цель
- Переместить задачу (изменить родительскую задачу)
- Установить/снять якорь задачи
- Управлять ручными представлениями, в которых находится задача
- Настроить повторяющуюся задачу
- Удалить / (в ручном представлении) удалить из представления
