# Функции списка задач

Список задач содержит несколько удобных функций. После применения фильтров порядок задач нельзя изменить вручную — доступна только автоматическая сортировка, но вы можете задать правила сортировки.

## Показывать только листовые узлы

Показывает только задачи без подзадач. Обычно это минимальные единицы исполнения.

## Сортировка по приоритету

Быстрая сортировка по приоритету. Доступно только при отключённой автоматической сортировке.

![sort](/steps/tasklist/sort_by_priority.png)

## Сортировка задач в режиме автоматической сортировки

В режиме автоматической сортировки есть кнопка настроек сортировки. Нажмите кнопку, чтобы переключить порядок (по возрастанию/по убыванию), и выберите измерение сортировки в выпадающем списке.

![sort](/steps/tasklist/sort.png)
