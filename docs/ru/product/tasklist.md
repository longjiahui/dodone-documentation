## Сортировка по приоритету

Быстрая сортировка по приоритету задачи. Доступно только при отключенной автоматической сортировке.

![sort](/steps/tasklist/sort_by_priority.png)

## Сортировка задач в режиме автоматической сортировки

В режиме автоматической сортировки есть кнопка настройки сортировки. Нажмите кнопку, чтобы установить порядок по возрастанию/убыванию, и выберите из выпадающего списка, чтобы изменить измерение сортировки.

![sort](/steps/tasklist/sort.png)
