# Представление задач

Представления задач позволяют настраивать запросы задач и не хранят сами данные задач. Они делятся на автоматические и ручные представления. Автоматические представления имеют логику поиска и автоматически ищут по списку задач, а ручные представления требуют ручного добавления задач, подобно функции `теги`.

## Свойства представления задач

- Название
- Иконка — Иконка ручного представления будет отображаться на задачах, которым назначено это представление. Иконка по умолчанию для ручных представлений — метка, для автоматических — робот.
- Описание
- Тип
  - Автоматическое
  - Ручное
- Логика автоматического представления
  - Скрипт
  - Логика настройки с GUI-помощью
