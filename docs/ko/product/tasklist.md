작업 목록에는 편리한 기능이 있습니다. 필터 기능을 사용한 후에는 작업 목록 순서를 수동으로 조정할 수 없으며 자동 정렬만 사용할 수 있지만 정렬 규칙은 구성할 수 있습니다.

## 리프 노드 작업만 표시

하위 작업이 없는 작업만 표시합니다. 이렇게 하면 일반적으로 가장 작은 실행 단위인 최하위 작업이 필터링됩니다.

![only-child](/steps/tasklist/only_child1.png)
![only-child](/steps/tasklist/only_child2.png)

## 작업 필터링

특정 조건에 따라 작업을 검색합니다.

![filter](/steps/tasklist/filter.png)

### 작업 필터링 취소

조건을 클릭하여 필터링 취소

![cancel-filter](/steps/tasklist/cancel_filter.png)

![show-done](/steps/tasklist/show_done1.png)

## 우선순위로 정렬

작업 우선순위로 빠르게 정렬합니다. 자동 정렬을 사용하지 않는 경우에만 사용 가능합니다.

![sort](/steps/tasklist/sort_by_priority.png)

## 자동 정렬 모드에서 작업 정렬

자동 정렬 모드에는 정렬 설정 버튼이 있습니다. 버튼을 클릭하여 오름차순/내림차순을 설정하고 드롭다운에서 선택하여 정렬 차원을 변경합니다.

![sort](/steps/tasklist/sort.png)
