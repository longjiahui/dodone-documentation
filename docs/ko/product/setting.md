데이터베이스는 모든 관련 데이터를 저장하는 파일입니다. 새 데이터베이스를 생성하면 완전히 새로운 데이터 공간이 만들어집니다. 이전 데이터베이스 파일이 너무 커서 느려지면 새 파일을 생성해 보세요.

#### 데이터베이스 삭제

:::warning
데이터베이스 삭제는 복구할 수 없습니다. 신중히 진행하세요.
:::

현재 사용 중인 데이터베이스는 삭제할 수 없습니다. 삭제해야 하는 경우, 먼저 빈 데이터베이스를 생성하고 전환한 후 삭제를 진행하세요.
