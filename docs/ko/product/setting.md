## 데이터베이스 설정

데이터베이스는 관련된 모든 데이터를 저장하는 파일입니다. 새 데이터베이스를 만드는 것은 새 데이터를 저장하기 위한 완전히 새로운 데이터 공간을 만드는 것과 같습니다. 이전 데이터베이스 파일이 너무 커지고 느려지면 새 파일을 열어보세요.

#### 데이터베이스 삭제

:::warning
데이터베이스 삭제는 되돌릴 수 없습니다. 주의해서 진행하세요.
:::

현재 사용 중인 데이터베이스는 삭제할 수 없습니다. 삭제해야 하는 경우 먼저 빈 데이터베이스를 만들고 새 데이터베이스로 전환한 다음 삭제를 진행하세요.
