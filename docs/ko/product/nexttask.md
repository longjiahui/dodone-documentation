dodone의 반복 작업은 다른 소프트웨어 도구와 다를 수 있습니다. 더 정확하게는 "다음 작업"입니다. 향후 반복 작업은 미리 생성되지 않고 현재 작업을 완료할 때 생성하라는 메시지가 표시됩니다. 예를 들어 매일 반복 작업을 설정하면 작업을 완료할 때 다음 날로 설정된 날짜로 다음 작업을 생성하라는 메시지가 표시됩니다.

상위 작업에 반복을 설정하면 다음 작업을 생성하라는 메시지가 표시될 때 하위 작업이 자동으로 복사됩니다.

상위 작업과 하위 작업 모두 반복 설정을 가질 수 있으며 모두 위에서 설명한 규칙을 따릅니다.

## 반복 작업 속성

- a - N일마다 반복
- b - 주기당 휴식 간격. 2로 설정하면 첫 번째는 1일 휴식하고 두 번째는 휴식하지 않습니다. 3으로 설정하면 첫 번째는 1일 휴식하고 두 번째는 2일 휴식하고 세 번째는 휴식하지 않습니다.
- endDate - 종료 날짜. 종료 날짜 이후에는 다음 작업을 생성하라는 메시지가 더 이상 표시되지 않습니다.
