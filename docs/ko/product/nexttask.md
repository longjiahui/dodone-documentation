# 반복 작업

dodone의 반복 작업은 다른 도구와 약간 다를 수 있습니다. 정확히 말하면 "다음 작업"입니다. 향후 반복 작업은 미리 생성되지 않으며 현재 작업을 완료할 때 다음 작업을 생성하라는 알림이 표시됩니다. 예를 들어 매일 반복 작업을 설정하면 해당 작업을 완료할 때 다음 작업을 생성하라는 메시지가 표시되며, 시간은 다음 날로 설정하도록 제안됩니다.

부모 작업에 반복을 설정하면 다음 작업을 생성할 때 하위 작업이 자동으로 복사됩니다.

부모 및 자식 작업 모두 하위 작업을 설정할 수 있으며, 위에 설명된 규칙을 따릅니다.

## 반복 작업 속성

- a: 며칠마다 반복할지
- b: 각 사이클의 휴식 간격. 2로 설정하면 첫 번째는 1일 휴식, 두 번째는 휴식 없음; 3으로 설정하면 첫 번째는 1일 휴식, 두 번째는 2일 휴식, 세 번째는 휴식 없음.
- endDate: 종료일. 이 날짜가 지나면 다음 작업을 생성하라는 알림이 더 이상 나타나지 않습니다.
