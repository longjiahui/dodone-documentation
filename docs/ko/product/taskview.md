# 작업 보기

작업 보기는 작업 쿼리를 사용자 정의할 수 있으며 작업 데이터를 실제로 저장하지 않습니다. 자동 보기와 수동 보기가 있습니다. 자동 보기는 작업 검색 로직을 가지며 작업 목록을 자동으로 검색합니다. 수동 뷰는 수동으로 작업을 설정해야 하며 `태그` 기능과 유사합니다.

## 작업 보기 속성

- 이름
- 아이콘 — 수동 작업 보기의 아이콘은 해당 보기가 설정된 작업에 표시됩니다. 수동 보기의 기본 아이콘은 태그, 자동 보기의 기본 아이콘은 로봇입니다.
- 설명
- 유형
  - 자동
  - 수동
- 자동 뷰 로직
  - 스크립트
  - GUI 보조 설정 로직
