# 작업

작업은 애플리케이션에서 가장 기본이 되는 데이터입니다.

## 작업 속성

| 필드          | key         |
| ------------- | ----------- |
| id            | id          |
| 내용          | content     |
| 설명          | description |
| 정렬 순서     | sortOrder   |
| 우선순위      | priority    |
| 작업량        | factor      |
| 완료 상태     | state       |
| 그룹 id       | groupId     |
| 상위 작업 id  | parentId    |
| 목표          | target      |
| 목표 유형     | targetType  |
| 시작일        | startAt     |
| 종료일        | endAt       |
| 완료 시간     | doneAt      |
| 생성 시간     | createdAt   |
| 업데이트 시간 | updatedAt   |

## 작업 동작

모든 작업 동작은 작업 툴바에 있습니다. 작업의 빈 영역 또는 오른쪽의 더보기 아이콘을 클릭하면 볼 수 있습니다.

![task-tools](/steps/task/task_tools.png)

왼쪽에서 오른쪽 순서로:

- 작업 내용 수정
- 하위 작업 편집
- 작업 설명 수정
- 작업 시간 수정
- 작업량 수정
- 우선순위 수정
- 목표 설정
- 작업 위치 이동 (상위 작업 변경)
- 작업 앵커 설정/해제
- 이 작업이 속한 수동 뷰 관리
- 반복 작업 설정
- 삭제 / (수동 뷰에서는) 뷰에서 제거

(이미지 생략)
