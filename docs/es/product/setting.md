## Configuración de la base de datos

Una base de datos es un archivo que guarda todos los datos relevantes. Crear una nueva base de datos equivale a crear un espacio de datos completamente nuevo para almacenar nuevos datos. Si el archivo de la base de datos anterior se vuelve demasiado grande y provoca lentitud, puede intentar crear un archivo nuevo.

#### Eliminar base de datos

:::warning
La eliminación de la base de datos no se puede deshacer. Proceda con precaución.
:::

No se puede eliminar la base de datos actualmente en uso. Si necesita eliminarla, cree primero una base de datos vacía y cambie a la nueva antes de realizar la eliminación.
