# Vista de tareas

Las vistas de tareas permiten personalizar las consultas de tareas y no almacenan realmente los datos de las tareas. Se dividen en vistas automáticas y vistas manuales. Las vistas automáticas tienen lógica de búsqueda y buscarán automáticamente en la lista de tareas, mientras que las vistas manuales requieren que las tareas se establezcan manualmente, similar a una `etiqueta`.

## Propiedades de la vista de tareas

- Nombre
- Icono — El icono de una vista manual se mostrará en las tareas a las que se haya asignado esa vista. El icono predeterminado de la vista manual es una etiqueta; el de la vista automática es un robot.
- Descripción
- Tipo
  - Automática
  - Manual
- Lógica de vista automática
  - Script
  - Configuración lógica asistida por GUI
