# Ventana "En curso"

La ventana "En curso" muestra la tarea en la que estás trabajando actualmente y tiene dos modos:

- Establecer una tarea específica para mostrar en la ventana "En curso"
- Leer automáticamente el calendario para obtener la tarea actual
