# Funciones de la lista de tareas

La lista de tareas tiene algunas funciones convenientes. Después de usar filtros, el orden de la lista no se puede ajustar manualmente y solo se puede usar la ordenación automática, pero se pueden establecer reglas de ordenación.

## Mostrar solo nodos hoja

Muestra solo las tareas que no tienen subtareas. Estas suelen ser las unidades mínimas de ejecución.

![only-child](/steps/tasklist/only_child1.png)
![only-child](/steps/tasklist/only_child2.png)

## Filtrado de tareas

Buscar tareas según condiciones específicas.

![filter](/steps/tasklist/filter.png)

### Cancelar filtrado

Haga clic en la condición para cancelar el filtro

![cancel-filter](/steps/tasklist/cancel_filter.png)

## Mostrar tareas completadas

![show-done](/steps/tasklist/show_done1.png)
![show-done](/steps/tasklist/show_done2.png)

### Ver la lista de tareas completadas

![show-done](/steps/tasklist/show_done_list.png)

## Ordenar por prioridad

Ordenar rápidamente por prioridad. Solo disponible cuando no se usa la ordenación automática.

![sort](/steps/tasklist/sort_by_priority.png)

## Ordenación en modo automático

En el modo de ordenación automática hay un botón de configuración de ordenación; pulse el botón para establecer ascendente/descendente y seleccione en el desplegable la dimensión de ordenación.

![sort](/steps/tasklist/sort.png)
