# Tarea

Las tareas son los datos más básicos de la aplicación.

## Propiedades de la tarea

| Campo                  | key         |
| ---------------------- | ----------- |
| id                     | id          |
| contenido              | content     |
| descripción            | description |
| orden                  | sortOrder   |
| prioridad              | priority    |
| carga de trabajo       | factor      |
| estado de finalización | state       |
| id del grupo           | groupId     |
| id de la tarea padre   | parentId    |
| objetivo               | target      |
| tipo de objetivo       | targetType  |
| fecha de inicio        | startAt     |
| fecha de fin           | endAt       |
| tiempo de finalización | doneAt      |
| creado                 | createdAt   |
| actualizado            | updatedAt   |

## Operaciones de tarea

Todas las operaciones de tarea están en la barra de herramientas de la tarea. Haga clic en el área vacía de una tarea o en el icono de más a la derecha para verlas.

![task-tools](/steps/task/task_tools.png)

De izquierda a derecha:

- Editar contenido de la tarea
- Editar subtareas
- Editar descripción
- Editar tiempo
- Editar carga de trabajo
- Editar prioridad
- Establecer objetivo
- Mover la tarea (cambiar tarea padre)
- Establecer/quitar ancla de tarea
- Administrar vistas manuales en las que se encuentra la tarea
- Configurar tarea recurrente
- Eliminar / (en vista manual) quitar de la vista
