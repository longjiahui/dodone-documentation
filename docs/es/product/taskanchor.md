# Ancla de tarea

Cuando se crean jerarquías de subtareas de 3, 4 niveles o más, la gestión de tareas puede volverse complicada. La creación de anclas puede resolver este problema. Las anclas son similares a los marcadores: al añadir un ancla, puedes acceder directamente a la tarea anclada desde el grupo.
